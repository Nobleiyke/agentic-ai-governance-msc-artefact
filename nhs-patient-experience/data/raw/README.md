# Raw data

Place downloaded NHS files here. **Do not commit** large or sensitive microdata to git.

## Suggested naming

```
ons_health_insight_wave22_2026.xlsx
fft_inpatient_december_2024.xlsx
npsp_adult_inpatient_national_tables_2022.ods
gpps_national_tables_2025.xlsx
```

## Week 1 minimum download set

1. One file from [NHS Surveys data library](https://nhssurveys.org/data-library/) (e.g. Adult Inpatient national tables)
2. One month from [NHS England FFT](https://www.england.nhs.uk/statistics/statistical-work-areas/friends-and-family-test-statistics/)
3. Latest wave from [ONS Experiences of NHS healthcare](https://www.ons.gov.uk/peoplepopulationandcommunity/healthandsocialcare/healthcaresystem/datasets/experiencesofnhshealthcareservicesinengland)

Record download date and URL in `../processed/DATA_DICTIONARY.md`.
